# ADAPTLEARN - Personalized Learning AI Agent 
Adaptive learning platform for students with personalized performance insights, weak-topic analysis, and tailored study recommendations.

A web-based platform that evaluates studentsâ€™ quiz performance, identifies weak topics, and provides personalized study recommendations with multilingual support.

Overview

It is designed to help students improve their academic performance by turning quizzes into actionable learning insights. It evaluates quiz answers, identifies weak subtopics, suggests study materials, and tracks progress over time. The system supports Physics, Chemistry, and Biology, and provides a multilingual interface in English, Hindi, and Tamil for wider accessibility.

Key Features

Secure Authentication: Signup and login with credential storage

Class & Subject Selection: Quizzes tailored to Students

Adaptive Quiz Generation: Randomly selects 10 questions per quiz from a pool of 20

Automated Performance Evaluation: Calculates scores and identifies weak subtopics

Personalized Study Recommendations: Provides YouTube links, study materials, and daily study schedules

Progress Tracking: Compares current and past quiz performances to measure improvement or decline.

Multilingual Support: Interface available in English, Hindi, and Tamil.

Technology Stack

Frontend: HTML, CSS, JavaScript

Backend: Node.js, Express

Database: SQLite
